Cell Phone Repair

This project was my final exam for a 12-week python course I did through BCIT. For this final I had 2 hours to make a cell phone repair shop's system that takes customer information and writes it to a file including their contact details and cell phone problem. It then allows you to search the "database" to find phones by serial # or name of customer. This project currently just runs in the terminal and uses number inputs to navigate the options. I will be updating this as a stand-alone application with a UI as my learning grows.

Personal Projects

Ability stone Calculator

This project is to recreate the ability stone cutting interface for the game Lost Ark. This program has a base success rate of 55% and fail rate of 45%. Each time you succeed the success rate goes down 10% and fail rate goes up 10%, and the opposite for when you fail. The goal is to get as many successes on the blue rows and as few as possible on the red row. This is a personal project I decided to work on because I thought it would be interesting to create the constantly changing probability with each success or fail and would be good practice for my coding skills.
